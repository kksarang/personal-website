# Sarang Rajan - Personal Portfolio

A modern, responsive personal portfolio website built with React, Vite, and Tailwind CSS.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository or navigate to the project folder.
2. Install dependencies:
   ```bash
   npm install
   ```

### Development

Start the development server:
```bash
npm run dev
```

### Build

Build the project for production:
```bash
npm run build
```

## 🛠️ Tech Stack
- **React.js**: Frontend library.
- **Vite**: Build tool and dev server.
- **Tailwind CSS**: Utility-first CSS framework.
- **Lucide React**: Icon library.

## 📝 Deployment

This project is ready to be deployed on **Netlify** or **Vercel**.
- For **Netlify**: Drag and drop the `dist` folder after running `npm run build`.
- For **Vercel**: Connect your GitHub repository and it will auto-detect the Vite configuration.
